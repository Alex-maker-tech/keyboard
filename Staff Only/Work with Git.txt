Работа с Git:
1. git init - создание в выбранной папке папки под названием .git
2. git add <file> / git add -A - добавление определенного файла / 
				 добавление всех файлов
3. git remote add origin <путь_к_репозиторию> - подключение к удаленному репозиторию
   <путь_к_репозиторию> - https://github.com/Alex-maker-tech/<название_репозитория>.git
4. git commit -m "<текст_коммита>" - создание коммита (фиксации изменений)
5. git branch -M main - создание ветки (branch) под названием main
6. git push origin main - отправка изменений на сервер
*. git clone <путь_к_репозиторию> - создание локального репозитория на компьютере,
	скачивание репозитория на компьютер
7. git pull origin main - скачивание изменений в репозитории
**. git branch - вывод доступных веток (активная помечена *)
***. git checkout <название_ветки> - переключение на указанную ветку
****. git merge <название_ветки> - объединить указанную ветку с активной
*****. git branch -d <название_ветки> - удалить указанную ветку
#. git log - вывод списка всех коммитов
##. git show <идентификатор_коммита> - вывод того, что нового появилось в коммите
